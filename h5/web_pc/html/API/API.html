debounce

解释：

防止高频触发

方法：

window.common.debounce(fun,times);

参数：

fun|Function
times|延迟触发时间(ms)

注意事项：

如果fun是一个动画，那么动画时间要小于延迟触发时间，否则无效

animation

解释：

优化动画

方法：

window.common.animation({times:'',frequency:''},fun);

参数：

times|动画时间(ms)
frequency|动画次数
fun|Function

注意事项：

fun为一个周期的动画
